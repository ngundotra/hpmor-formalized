# Logic

HPMOR Chapters: 27

## Goal

Formalize the logical and computational aspects of HPMOR's magical
concepts — particularly where magic implies computational or
information-theoretic properties that the story treats casually.

## What's here

| File | What it does | Tier | Finding |
|------|-------------|------|---------|
| Occlumency.lean | Models mind-shielding as cryptographic indistinguishability. Proves perfect Occlumency requires identical distributions (simulation paradox). | 2 (borderline) | #10 |

## Findings so far

- **Occlumency**: Perfect mind-shielding requires the fake mental state to be distributionally identical to the real one — a simulation paradox. The info-theoretic vs computational distinction is real but the core observation is somewhat definitional.
