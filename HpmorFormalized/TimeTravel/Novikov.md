In HPMOR, Time-Turners let you send information back in time, but only if the result is self-consistent -- no paradoxes allowed. We modeled this in two ways: first, as a function that maps universe-states to universe-states, and second, as a directed graph of cause-and-effect events that has no loops. For the state-machine model, we proved that on any finite set of states, such a function must eventually loop back to where it started (using the pigeonhole principle -- if you keep stepping through states and there are only finitely many, you must revisit one). We also proved that if the cause-and-effect graph has no cycles, then a consistent assignment of states to events always exists. This confirms the HPMOR claim: self-consistent time loops are not just possible, they are mathematically guaranteed, as long as the number of possible states is finite. The key hidden assumption the formalization makes explicit is finiteness -- if the state space were infinite (like the natural numbers), a function could keep producing new states forever and never loop back.
