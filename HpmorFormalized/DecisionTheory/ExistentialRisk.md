In HPMOR Chapters 43-47, Harry reasons about Dementors as an existential threat: any probability of total annihilation times negative infinity utility equals negative infinity, so preventing it trumps all other considerations. We formalized this using extended real numbers (EReal) and proved Harry's calculation is correct — but also proved it's useless. If every available action has some tiny chance of catastrophe, then every action scores negative infinity and the framework can't distinguish between them. Harry's actual reasoning uses a different system: first minimize the probability of annihilation, then optimize everything else. This is called lexicographic preferences, not expected utility. The formalization caught Harry using the wrong framework to get the right answer.
